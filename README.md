# Presence-monitor


### Requirements

 - Raspberry Pi
 - HC-SR501
 - Python 3

### Connection


| PIR PIN | Raspberry Pi PIN |
|---------|------------------|
| VCC     | 5V (4) |
| GND     | GND (pin 6)      |
| OUT     | GPIO4 (pin 7)    |



```
sudo apt update
sudo apt install python3-rpi.gpio
```
